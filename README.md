# Trace Homebrew tap

Homebrew tap for **tracecli**, the command-line tool for [Trace](https://traceapp.info).

```sh
brew install trace-app-mac/tap/tracecli
```

`tracecli` reads your past Trace meetings and summaries and controls the live app
from the command line, for people in a terminal and for AI agents. Run
`tracecli --help` after installing.

The formula and the release binaries here are published automatically from the
Trace project. This tap holds only the compiled (arm64) binary and the formula,
no source.
